WSO2 Carbon Kernel ${carbon.version} Release Notes

${buildNumber}

WSO2 Carbon Kernel ${carbon.version} is the core of the next-generation WSO2 Carbon platform. It is completely architected from the ground up with the latest technologies and patterns. Additionally, the Carbon Kernel is now a lightweight, general-purpose OSGi runtime specializing in hosting servers, providing key functionality for server developers. The result is a streamlined and even more powerful middleware platform than ever before.

WSO2 Carbon Kernel is an open source project and it is available under the Apache Software License (v2.0) .

Key Features

  • Java 8 Support
  • Carbon Component Startup Order Resolver
  • Carbon Tools - Jar To Bundle Converter and Dropins Installer
  • Equipped with Eclipse Luna SR2 OSGi Framework
  • OSGi-Lib Support for OSGi Ready Bundles
  • Carbon Launcher
  • Centralized Logging Framework with Log4j 2.0 as the Backend
  • Transport Management
  • Pluggable Runtime Management
  • OSGi Test Framework
  • Audit Log Mechanism
  • JMX support
  • Carbon Context API
